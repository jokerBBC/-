═════════════════ [ 前置准备：恢复纯净与定位目录 ] ═════════════════════════

⚠️ 【重要提示：关于旧版模组】
如果你以前安装过的旧版覆盖原文件型魔改（直接替换文件或修改 plugins.js 的版本），
请勿继续使用！所有模组现已全面适配 ModLoader.js（Mod管理器），与其一并打包。
为防止新旧文件冲突导致游戏崩溃，安装前【必须】执行以下纯净恢复操作：

▶ 步骤一：在 Steam 中验证游戏完整性
  1. 打开 Steam 客户端，在「库」中找到本游戏。
  2. 右键点击游戏名称，选择「属性」。
  3. 在左侧菜单中点击「已安装文件」。
  4. 点击「验证游戏文件完整性」按钮。
  5. 等待 Steam 自动扫描并下载缺失/修复被篡改的文件（进度条走完即可）。

▶ 步骤二：在 Steam 中打开游戏目录
  1. 在 Steam「库」中右键点击本游戏。
  2. 选择「管理」 -> 「浏览本地文件」。
  3. 此时弹出的文件夹窗口，就是【游戏文件夹】（里面有 Game.exe）。



═════════════════════════════ [ 第一步：部署文件 ] ═══════════════════════════

找到你安装游戏的【游戏文件夹】（里面有一个 Game.exe 图标）。

👉 将我提供的「index.html」文件及「js文件夹」，一并拖入【游戏文件夹】中，选择覆盖原文件。

📌 文件摆放层级必须严格符合下方结构（注意区分层级关系）：

[游戏文件夹]
  ├─ Game.exe
  ├─ index.html            👈 覆盖原文件
  └─ js 文件夹 (打开它)
       └─ mods 文件夹 (打开它)
             ├─ img 文件夹 (打开它) 
             │	   ├─ 模组管理.png 👈 这里可以替换喜欢的图标，名字和格式一致
             │    └─ 其他mod.png（整合包内的独立插件有图片的，都做了路径适配）
             ├─  ModLoader.js       👈 放在这里(以后下载的 mod.js文件，全部放在这里,覆盖型别放进来)
             └─  其他mod.js（删除这个文件夹里的插件时记得先在管理器里关闭该插件，不然会报错）

═════════════════════════════ [ 常见问题排查 ] ═══════════════════════════

Q: 修改了模组状态但进游戏没反应？
A: 请仔细阅读上方红色的【生效规则】，必须按 F5 刷新页面。

Q: 模组文件放进去，但是管理器里扫描不到？
A: 确认文件后缀名是 .js，并且确认放在了 js/mods 文件夹里，
   而不是 js/mods/img 文件夹里。

Q: 更新后进游戏，使用某些插件功能，游戏报错？
A: 如果游戏作者发布了游戏更新，可能会改写 「plugins文件夹」的原本的js 文件。
    打开游戏文件夹\js\plugins 按修改日期排序下确认：最新更新的插件是否是报错那个。
    魔改插件基于更新前的插件做的，代码不匹配失效了，mod管理器界面关掉它。

Q: 更新后进游戏，看得到模组管理的图标，但所有mod失效了？
A: 如果游戏作者发布了游戏更新，可能会覆盖掉 plugins.js 文件。
   这种情况请在次打开mod管理器界面，随便开关一个mod，F5刷新。

Q: 更新后进游戏，完全没有看到模组管理的图标？
A: 如果游戏作者发布了游戏更新，可能会覆盖掉 index.html 文件。
   这种情况请参考下方的【手动修复指南】，重新注入一行代码即可。


═══════════════════ [ 兜底：手动修复 index.html ] ══════════════════════════
(仅在游戏更新导致模组图标消失时使用)

👉 鼠标右键点击游戏文件夹里的 index.html，选择「打开方式」 -> 「记事本」。

👉 在文件中按下 Ctrl+F 搜索这段代码：
   <script src="js/main.js"></script>

👉 在它的【正上方】，新起一行，粘贴这段代码：
   <script src="js/ModLoader.js"></script>

📌 修改后的效果必须严格符合下方示例：
   ...前面的代码不用管...
   
   <script src="js/ModLoader.js"></script>
   <script src="js/main.js"></script>
   
   ...后面的代码不用管...

👉 粘贴确认无误后，保存文件 (Ctrl+S) 并关闭记事本，重启游戏即可修复。
